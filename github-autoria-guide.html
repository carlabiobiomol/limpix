Como manter sua autoria

⚠️

Importante: no Brasil, direitos autorais sobre software e método são protegidos pela Lei 9.610/98 e Lei 9.609/98, desde o momento da criação — sem necessidade de registro. Mas registrar prova a data e facilita qualquer disputa futura.

📁
Commit no GitHub
O histórico de commits do GitHub tem data e hora registrada em servidor. É prova técnica de quando você criou. Seu primeiro commit já documenta a autoria.
📜
Licença no repositório
O arquivo LICENSE na raiz do repositório define o que outros podem ou não fazer com seu código. Use a licença proprietária incluída neste projeto.
🏛️
INPI — Registro de marca
Registre "Limpix" como marca no INPI (inpi.gov.br). Custa ~R$355 e protege o nome comercialmente em território nacional por 10 anos.
📧
E-mail datado
Envie para seu próprio e-mail um arquivo .zip com todo o projeto descrevendo o que é. Data e hora do servidor de e-mail viram prova adicional.
✅

O mais importante agora: faça o primeiro commit hoje. O carimbo de data do GitHub é reconhecido juridicamente como evidência de criação. Registro no INPI pode vir depois, quando o projeto estiver gerando receita.

🐙

Criar o repositório no GitHub

1
Crie sua conta no GitHub
Acesse github.com e crie uma conta. Escolha o username com cuidado — vai aparecer no link do seu projeto. Sugestão: @liah-limpix ou @limpix-app
2
Crie o repositório
Clique em "New repository". Configure assim:
Nome: limpix Visibilidade: Public (aparece no portfólio) Descrição: "Organização inteligente do Google — método, scripts e automações" README: ✅ Adicionar automaticamente Licença: Nenhuma (vamos adicionar a nossa)
3
Monte a estrutura de pastas
Clique em "Add file → Create new file" e crie os arquivos da estrutura. Ou use o Git no terminal:
# No terminal (se tiver Git instalado) git clone https://github.com/SEU_USUARIO/limpix cd limpix # Crie as pastas mkdir metodo scripts templates marketing docs # Copie os arquivos baixados aqui # Depois suba tudo: git add . git commit -m "feat: estrutura inicial do projeto Limpix" git push origin main
4
Configure o perfil do repositório
No repositório, clique na engrenagem ⚙️ ao lado de "About" e preencha:
  • Descrição: "Serviço de organização inteligente do Google — Gmail, Drive e Fotos"
  • Website: link do seu Instagram ou landing page
  • Topics (tags): gmail google-drive automation apps-script saas brasil
5
Ative o GitHub Pages (opcional mas recomendado)
Transforma o README em uma página web pública. Vá em Settings → Pages → Branch: main → Save. Seu projeto vai ter um link tipo seuusuario.github.io/limpix
💡

Adicione o link do seu GitHub no Instagram e na bio de todas as redes. Um repositório organizado e bem documentado passa credibilidade imediata — especialmente para clientes B2B e desenvolvedores que possam querer colaborar.
